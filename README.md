## 📖 Overview

- 🌱 Lawn Care  
- 🍂 Seasonal Cleanup  

### 📍 Service Areas

- Waterford, CT  
- New London, CT  
- Norwich, CT

## 🎨 Customization Guide

### 🔤 Change Business Name

Edit:

```html
<title>Patrick's Services</title>
<div class="logo">Patrick's Services</div>
```

---

### 🗺️ Edit Service Areas

```html
<div class="location-card">
    <h3>City Name</h3>
    <p>Service description.</p>
</div>
```

---

### 🧰 Edit Services

```html
<div class="service">
    <img src="IMAGE_URL" alt="Description">
    <div class="service-content">
        <h3>Service Name</h3>
        <p>Short description.</p>
    </div>
</div>
```

---

### 🌑 Change Theme Colors

Modify the CSS variables at the top of the file:

```css
:root {
    --bg-main: #0a0a0a;
    --bg-section: #111111;
    --bg-card: #161616;
    --bg-header: #050505;
    --text-main: #f5f5f5;
    --text-muted: #9a9a9a;
    --accent: #ffffff;
    --border: #222222;
}
```
